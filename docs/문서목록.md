# SAR 관련 문서 목록

---

## 1. 문서들 관계 한눈에 보기

```
┌─────────────────────────────────────────────────────────────┐
│                      우리가 공부할 문서들                     │
└─────────────────────────────────────────────────────────────┘

    ┌─────────────┐
    │  RG 1.206   │  ← "SAR 이렇게 써라" (신청자용 가이드)
    └─────────────┘
           │
           │ 참고해서 작성
           ▼
    ┌─────────────┐
    │ NuScale DCA │  ← 실제 SAR 샘플 (이렇게 쓰면 됨)
    └─────────────┘
           │
           │ NRC가 검토
           ▼
    ┌─────────────┐
    │  SRP / DSRS │  ← "이 기준으로 평가한다" (NRC용 매뉴얼)
    └─────────────┘
```

---

## 2. 각 문서가 뭔지

### RG 1.206 (Regulatory Guide 1.206)

```
┌─────────────────────────────────────────────┐
│  RG 1.206                                   │
│  "신규 경수로 통합 인허가 신청서 작성법"       │
├─────────────────────────────────────────────┤
│  - 누가 봐? → 원전 짓고 싶은 회사            │
│  - 뭐가 있어? → SAR 각 챕터에 뭘 써야하는지  │
│  - 적용 대상 → 10 CFR Part 52 (통합인허가)  │
└─────────────────────────────────────────────┘
```

### SRP (Standard Review Plan, NUREG-0800)

```
┌─────────────────────────────────────────────┐
│  SRP (NUREG-0800)                           │
│  "대형 경수로 SAR 검토 매뉴얼"               │
├─────────────────────────────────────────────┤
│  - 누가 봐? → NRC 직원                      │
│  - 뭐가 있어? → 검토 기준, 체크리스트        │
│  - 적용 대상 → 대형 원전 (LWR)              │
│  - 참고: 꼭 따를 필요 없음 (대안 시 설명)    │
└─────────────────────────────────────────────┘
```

### DSRS (Design-Specific Review Standard)

```
┌─────────────────────────────────────────────┐
│  DSRS                                       │
│  "SMR 전용 검토 매뉴얼"                      │
├─────────────────────────────────────────────┤
│  - 누가 봐? → NRC 직원                      │
│  - 뭐가 있어? → SMR 특화 검토 기준          │
│  - 적용 대상 → 소형모듈원전 (NuScale 등)    │
│  - 왜 따로? → SMR은 대형원전이랑 많이 달라서 │
└─────────────────────────────────────────────┘
```

### NuScale DCA (Design Certification Application)

```
┌─────────────────────────────────────────────┐
│  NuScale DCA                                │
│  "NuScale이 실제로 제출한 SAR"               │
├─────────────────────────────────────────────┤
│  - 이게 왜 중요? → 실제 작성 샘플!           │
│  - 뭘 배워? → SAR 작성 패턴, 형식           │
│  - 상태 → NRC 승인됨 (2020년)               │
└─────────────────────────────────────────────┘
```

## 5. SRP/SAR 챕터 구조 (19개)

SAR은 19개 챕터로 구성됨. SRP도 같은 구조로 되어있음.

```
┌────────────────────────────────────────────────────────────┐
│                    SAR / SRP 챕터 구조                      │
├────┬───────────────────────────────────────────────────────┤
│ Ch │ 제목                           │ 뭔 내용?             │
├────┼───────────────────────────────────────────────────────┤
│  1 │ Introduction                   │ 원전 소개            │
│  2 │ Site Characteristics           │ 부지 특성            │
│  3 │ Design of Structures...        │ 구조물/기기 설계     │
│  4 │ Reactor ⭐                     │ 원자로 (핵심!)       │
│  5 │ Reactor Coolant System         │ 냉각재 계통          │
│  6 │ Engineered Safety Features ⭐  │ 공학적 안전설비      │
│  7 │ Instrumentation and Controls   │ 계측제어             │
│  8 │ Electric Power                 │ 전력 계통            │
│  9 │ Auxiliary Systems              │ 보조 계통            │
│ 10 │ Steam and Power Conversion     │ 증기/발전 계통       │
│ 11 │ Radioactive Waste Management   │ 방사성폐기물 관리    │
│ 12 │ Radiation Protection           │ 방사선 방호          │
│ 13 │ Conduct of Operations          │ 운영 관리            │
│ 14 │ Initial Test Programs          │ 초기시험/ITAAC       │
│ 15 │ Transient and Accident ⭐      │ 사고 분석 (핵심!)    │
│ 16 │ Technical Specifications       │ 기술지침서           │
│ 17 │ Quality Assurance              │ 품질보증             │
│ 18 │ Human Factors Engineering      │ 인간공학             │
│ 19 │ Probabilistic Risk Assessment  │ 확률론적 위험평가    │
└────┴───────────────────────────────────────────────────────┘

⭐ = 핵심 챕터 (우선 공부)
```
