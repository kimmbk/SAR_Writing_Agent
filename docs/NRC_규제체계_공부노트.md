```

└─────────────────────────────────────────────────────────────┘
```

**NRC** = Nuclear Regulatory Commission (미국 원자력규제위원회)

## 2. 인허가 방식 2가지 (10 CFR)

**10 CFR** = Code of Federal Regulations, Title 10 (미국 원자력 규제 법률집)

```
                    10 CFR = 미국 원자력 규제 법률집

    ┌──────────────────────┐       ┌──────────────────────┐
    │     Part 50          │       │     Part 52          │
    │    (옛날 방식)        │       │    (요즘 방식)        │
    └──────────────────────┘       └──────────────────────┘
              │                              │
              ▼                              ▼
    ┌──────────────────────┐       ┌──────────────────────┐
    │  1단계: 건설허가      │       │  한방에: 통합인허가   │
    │        ↓             │       │  (설계+건설+운영)     │
    │  2단계: 운영허가      │       │                      │
    └──────────────────────┘       └──────────────────────┘
              │                              │
        시간 오래 걸림                    빠르고 효율적
        (중간에 변경 가능)               (미리 다 정해놓음)
```

### Part 50 vs Part 52 비교

| 항목 | Part 50 | Part 52 |
|------|---------|---------|
| 방식 | 2단계 (건설 → 운영) | 1단계 (통합인허가) |
| 특징 | 전통적, 유연함 | 효율적, 예측 가능 |
| 설계 변경 | 건설 중 변경 가능 | 미리 확정 필요 |

### GDC (General Design Criteria)

- **위치**: 10 CFR Part 50 Appendix A
- **내용**: 원전 설계 시 만족해야 하는 일반 기준
- **중요**: Part 52도 이 GDC를 따름

## 3. 신청자가 제출하는 문서: SAR

```
    원전 짓고 싶은 회사
           │
           │  "우리 원전 이렇게 안전해요"
           │   문서 제출
           ▼
    ┌─────────────────────────────────────┐
    │          SAR                        │
    │   (Safety Analysis Report)          │
    │       안전분석보고서                  │
    │                                     │
    │   - 원전 설계 설명                   │
    │   - 사고 분석                        │
    │   - 안전 시스템 설명                 │
    │   - ... (19개 챕터)                  │
    └─────────────────────────────────────┘
```

**SAR** = Safety Analysis Report (안전분석보고서)
- 원전 신청자가 NRC에 제출하는 핵심 문서
- "우리 원전이 왜 안전한지" 설명하는 문서

## 4. SAR 어떻게 쓰지? → RG (Regulatory Guide)

```
    신청자: "SAR 뭘 써야 해요?"
                │
                ▼
    ┌─────────────────────────────────────────────────────┐
    │              RG (Regulatory Guide)                  │
    │              = NRC가 만든 작성 가이드                │
    │                                                     │
    │   "각 챕터에 이런 내용 넣어라"                        │
    └─────────────────────────────────────────────────────┘
                │
        ┌───────┴───────┐
        ▼               ▼
    ┌─────────┐    ┌─────────┐
    │ RG 1.70 │    │RG 1.206 │
    │         │    │         │
    │ Part 50 │    │ Part 52 │
    │  용      │    │  용      │
    │ (1978)  │    │ (2007)  │
    └─────────┘    └─────────┘
```

**RG** = Regulatory Guide (규제 지침)
- NRC가 신청자에게 주는 가이드
- "SAR에 이런 내용을 이렇게 써라"

### RG 1.70 vs RG 1.206

| 항목 | RG 1.70 | RG 1.206 |
|------|---------|----------|
| 적용 | Part 50 (2단계 인허가) | Part 52 (통합인허가) |
| 발행 | 1978년 | 2007년 |
| 제목 | SAR 표준 형식 | 신규 경수로 통합 인허가 신청 |

## 5. NRC는 어떻게 검토해? → SRP / DSRS

```
    NRC 직원: "SAR 검토해야 하는데 기준이 뭐지?"
                │
                ▼
    ┌─────────────────────────────────────────────────────┐
    │         SRP / DSRS                                  │
    │         = NRC 직원용 검토 매뉴얼                     │
    │                                                     │
    │   "이 항목은 이렇게 평가해라"                         │
    │   "이 기준 만족하는지 확인해라"                       │
    └─────────────────────────────────────────────────────┘
                │
        ┌───────┴───────┐
        ▼               ▼
    ┌─────────┐    ┌─────────┐
    │   SRP   │    │  DSRS   │
    │         │    │         │
    │  대형   │    │  소형   │
    │  원전용  │    │ 원전용  │
    │ (LWR)  │    │ (SMR)   │
    └─────────┘    └─────────┘
```

### SRP (Standard Review Plan)

- **문서번호**: NUREG-0800
- **대상**: 대형 경수로 (LWR)
- **용도**: NRC 직원이 SAR 검토할 때 사용하는 매뉴얼
- **특징**: 필수는 아님! 대안 사용 시 충분한 설명 필요

### DSRS (Design-Specific Review Standard)

- **대상**: 소형모듈원전 (SMR) - 예: NuScale
- **용도**: SRP와 같지만 SMR 특화
- **이유**: SMR은 대형 원전과 설계가 많이 달라서 별도 기준 필요

## 7. 용어 정리

| 약어 | 풀네임 | 한글 | 설명 |
|------|--------|------|------|
| **NRC** | Nuclear Regulatory Commission | 미국 원자력규제위원회 | 미국 원전 인허가 담당 기관 |
| **CFR** | Code of Federal Regulations | 연방규정집 | 미국 법률 모음 |
| **SAR** | Safety Analysis Report | 안전분석보고서 | 신청자가 제출하는 문서 |
| **RG** | Regulatory Guide | 규제지침 | SAR 작성 가이드 |
| **SRP** | Standard Review Plan | 표준심사지침 | NRC 검토 매뉴얼 (대형원전) |
| **DSRS** | Design-Specific Review Standard | 설계별심사기준 | NRC 검토 매뉴얼 (SMR) |
| **LWR** | Light Water Reactor | 경수로 | 일반적인 대형 원전 |
| **SMR** | Small Modular Reactor | 소형모듈원전 | 작은 원전 (NuScale 등) |
| **GDC** | General Design Criteria | 일반설계기준 | 원전 설계 시 만족해야 할 기준 |

## 8. 핵심 요약 (시험용)

### 한줄 요약

| 문서 | 누가 보는지 | 한마디 |
|------|-----------|--------|
| **10 CFR** | 모두 | 법률 (지켜야 함) |
| **RG** | 신청자 | SAR 작성법 |
| **SRP/DSRS** | NRC 직원 | SAR 검토법 |
| **SAR** | 신청자→NRC | 실제 제출 문서 |

### 기억할 것

1. **Part 50** = 옛날 방식 (건설허가 → 운영허가)
2. **Part 52** = 요즘 방식 (통합인허가)
3. **RG 1.70** = Part 50용 SAR 가이드
4. **RG 1.206** = Part 52용 SAR 가이드
5. **SRP** = 대형원전 검토 기준
6. **DSRS** = SMR 검토 기준
7. **GDC** = Part 50에 있지만 Part 52도 따름
